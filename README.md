# potato

马铃薯，一个面向开发者的编码智能体产品。

这个仓库用于沉淀一个面向开发者的编码智能体产品。

当前方向是先验证核心智能体工作流：

- 使用 Pi 作为底层智能体执行引擎。
- 在 `core/` 中沉淀本项目自己的 AgentGateway、AgentOrchestrator、Pi 适配、权限、trace、diff 和工具边界。
- 先用 `cli/` 作为第一阶段交互壳。
- 让用户能看到智能体正在做什么。
- 清晰展示文件差异。
- 在核心工作流被验证前，暂缓桌面端 UI 投入。

项目笔记、架构决策、调研记录和实现计划统一存放在 [`wiki/`](wiki/README.md)。
